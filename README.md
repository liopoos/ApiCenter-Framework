# API Center Framework

### 什么是API Center？

欢迎体验API Center：[api.mayuko.cn](https://api.mayuko.cn/).

API Center集合了mayuko.cn站点下的所有api。在API Center，你可以选择你感兴趣的API，并使用它们。

------



### 这是什么？

如果你喜欢API Center的样式，这是提供了API Center的框架，你可以使用它。

------
